## apt-get命令

**更新包数据库：**

```sh
sudo apt-get update
```

**搜索包：**

```sh
apt-cache search 包名
```

**安装包：**

```sh
sudo apt-get install <package_name>
sudo apt-get install <package_name> --no-upgrade   #在不升级的情况下安装包
sudo apt-get install <package_name>=<version_number>   #安装特定版本的安装包
sudo apt-get install <package_1> <package_2> <package_3>  #安装多个包
```

**删除安装包：**

```sh
sudo apt-get remove <package_name>
sudo apt-get purge <package_name>
```

:warning:注意：

- `apt-get remove` 只删除包的二进制文件，它不会触及配置文件。
- `apt-get purge` 删除与包相关的所有内容，包括配置文件。

**更换apt源：**

1、查看系统版本号并备份：

```sh
lsb_release -c
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
```

2、更换源（18.04）：

```sh
vi /etc/apt/sources.list
```
